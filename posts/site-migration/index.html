Site Migration

2023-03-05 :: Dhruv Joshi :: 1 min read (165 words)
#migration 

Finally!

It has been a while since I wrote anything here.

Migrating this site’s theme had been on my checklist for a long time.

Initially, I had used tranquilpeak theme. But soon I found out that there were many features provided by that theme which I was never going to use. And I wanted to keep the design of the site to as minimal as possible.

For the past couple of days I have been surfing through hugo themes in search for that minimal theme. And the result is in front of you. I have migrated to the terminal theme.

Fortunately, there were very few post I had written so the migration process did not took that much time. But one thing is certain, that I am in love with this theme and will continue with it for a long term.

One final note though, I will be more active on publishing useful technical articles from now on.

See you in the next post soon!
